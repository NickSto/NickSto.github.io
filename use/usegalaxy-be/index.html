UseGalaxy.be

Public server:ELIXIR Belgium Galaxy Server
Scope:Genomics
Summary:General purpose genomics Galaxy server

Comments

  • A public Galaxy server with an emphasis on serving researchers in Belgium.
  • Real time status
  • Provides reference data for plant species from the PLAZA data warehouse.

User Support

  • Email: galaxy@elixir-belgium.org

Quotas

  • Free to use
  • 50 GB for any registered users (5GB for unregistered users)
  • Special requests contact galaxy@elixir-belgium.org

Citations

  • UseGalaxy.be platform for data-intensive research, Frederik Coppens, ELIXIR Belgium All Hands 2019
