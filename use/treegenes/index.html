TreeGenes

Public server:TreeGenes Galaxy
Scope:Domain specific
Summary:TreeGenes integrates genotype, phenotype, and environmental data that can be processed for association mapping and landscape genomics, with a focus on forest trees. This functionality is provided by Tripal Galaxy, a module for Tripal that provides seamless support for running Galaxy workflows directly in the site. Users can analyze data provided by TreeGenes or their own.

Comments

  • TreeGenes is utilizing the Tripal Galaxy API in order to provide seamless Galaxy workflow support to users on the site. Examples currently in development
  • CartograTree: Web-based map-driven module to provide association and landscape genomics. Uses Sambada
  • Orthoquery: Module to assess and visualize gene family evolution. Uses tools: VSEARCH, OrthoFinder, Diamond and TransDecoder

User Support

  • Accounts can be requested through the TreeGenes site

Quotas
