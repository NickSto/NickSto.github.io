BioCompute Object Galaxy

Public server:BioCompute Object server
Scope:Genomics
Summary:BioCompute Objects is a standard for communication of High-throughput Sequencing (HTS) analysis results, data set creation and curation, and bioinformatics verification protocols.

Comments

  • BioCompute is the IEEE 2791-2020 standard or bioinformatics pipelines/workflows in the High-Throughput Sequencing (HTS) space and other computational workflows. The BioCompute standard is supported by Google Cloud for workflow sharing and by the FDA Centers, CDER, CBER, and CFSAN, for regulatory submissions on a voluntary submitter basis.
