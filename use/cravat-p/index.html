CRAVAT-P

Container:CRAVAT-P Galaxy Docker
Scope:Tool Publishing
Summary:A Docker image containing a fully-operational Galaxy instance with pre-installed demonstration material for CRAVAT-P.

Comments

  • The CRAVAT-P Galaxy Docker instance includes a Galaxy Instance (version 17.09); CRAVAT-P submit, intersect, annotate, and retrieve Galaxy tool; CRAVAT-P visualization plugin; input files (i.e., VCF and proBED files); and a basic CRAVAT-P workflow.
