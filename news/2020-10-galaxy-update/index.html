Galaxy Platforms News

The Galaxy Platform Directory lists resources for easily running your analysis on Galaxy, including publicly available servers, cloud services, and containers and VMs that run Galaxy. Here's the recent platform news we know about:

[BRIDGE](/src/use/bridge/index.md)

BRIDGE (Biomolecular Reaction and Interaction Dynamics Global Environment) is a Computational Chemistry flavour of Galaxy - a webserver for preparing, simulating and analysing and visualising molecular simulations, especially molecular dynamics and free energies. Includes NAMD, GROMACS and CHARMM. Offers tutorials and a Gitter support channel.

[BioCompute Object](/src/use/biocompute-object/index.md)

Implements the BioCompute Objects standard for communication of High-throughput Sequencing (HTS) analysis results, data set creation and curation, and bioinformatics verification protocols within a Galaxy server. Extensive documentation and chat support are available.

[A Quick Tour of French Galaxy Instances](http://eepurl.com/hcRat1)

The 24 September IFB newsletter highlights 10 Galaxy resources in France (believe it or not, there are more), including a few public servers.

[PhagePromoter](/src/use/phagepromoter/index.md)

PhagePromoter predicts promoter sequences in phage genomes, using machine learning models. This server is hosted by the Centre of Biological Engineering, School of Engineering at the University of Minho — our first public server in Portugal.

Releases

[Pulsar 0.14.0](https://github.com/galaxyproject/pulsar/blob/master/HISTORY.rst)

Pulsar is a Python server application that allows a Galaxy server to run jobs on remote systems (including Windows) without requiring a shared mounted file systems. Unlike traditional Galaxy job runners - input files, scripts, and config files may be transferred to the remote system, the job is executed, and the results are transferred back to the Galaxy server - eliminating the need for a shared file system.
